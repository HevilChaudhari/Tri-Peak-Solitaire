# Tri-Peak Solitaire (Unity)

A classic Tri-Peak Solitaire card game developed in Unity. The game follows
standard Tri-Peak rules, featuring smooth card interactions, win/lose
conditions, and a clean gameplay flow.

## Features
- Classic Tri-Peak Solitaire rules
- Interactive card selection
- Draw pile and waste pile system
- Valid move checking based on card values
- Win and lose conditions
- Restart and new game functionality
- Simple and responsive UI

## Tech Stack
- Unity
- C#
- Unity UI (Canvas)
- ScriptableObjects (for card data)

## What This Project Demonstrates
- Card game logic and rule implementation
- Game state management
- UI-driven gameplay interactions
- Clean separation of gameplay systems
- Handling win/lose conditions

## Platform
- Android

## How to Play
1. Select cards that are one rank higher or lower than the current open card
2. Use the draw pile when no valid moves are available
3. Clear all face-up cards to win the game

## How to Run
1. Open the project in Unity
2. Select the desired platform (Android or PC)
3. Build and run the project
4. Start a new game and play

## Note
This project is a gameplay demo intended to showcase card game logic and UI
interaction. Visual assets are used for demonstration purposes only.
